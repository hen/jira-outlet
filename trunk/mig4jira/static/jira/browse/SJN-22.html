 Description   
Need a pluggable converter system. It would specify the object to create and sub-properties would be bean-properties.
ie)

org.osjava.converter.java.util.Date=DateConverter
org.osjava.converter.java.util.Date.leapyear=true
org.osjava.converter.java.util.Date.leapyear.type=java.lang.Boolean

or maybe not. Something like that I think, but the recursiveness there is scary.

org.osjava.converter.timezone=GMT

org.osjava.converter=DateConverter
DateConverter.converts=java.util.Date
DateConverter.timezone=GMT
